\# Yas Hesaplayici



Dogum tarihinden bugune kadar gecen sureyi yil, ay, gun cinsinden hesaplayan basit bir Python CLI uygulamasi.



\## Ozellikler



\- Dogum tarihinden buguene yil/ay/gun hesabi

\- Toplam yasanmis gun sayisi

\- Turkce arayuz, dis kutuphane gereksiz



\## Kurulum



Python 3.7+ gereklidir. Ek bir kutuphane yok.



\\`\\`\\`bash

git clone https://github.com/mesutakdogan20/yas-hesaplayici.git

cd yas-hesaplayici

python main.py

\\`\\`\\`



\## Kullanim
